# homebrew-lazyack

Homebrew tap for [lazyack](https://github.com/AngryCatKR96/lazyack) — global
hotkey to approve AI coding agent prompts via cmux, without switching focus.

## Install

```bash
brew install AngryCatKR96/lazyack/lazyack
```

The above is shorthand for:

```bash
brew tap AngryCatKR96/lazyack
brew install lazyack
```

To upgrade later:

```bash
brew upgrade lazyack
```

To remove:

```bash
brew uninstall lazyack
brew untap AngryCatKR96/lazyack
```

See the main [lazyack repository](https://github.com/AngryCatKR96/lazyack)
for usage, configuration, and `lazyack doctor` diagnostics.
